Introduced AutoTLS in libp2p(-py)

February 2, 2026 · Abhinav Agarwalla

Extended the TLS security module in libp2p(-py) to fetch CA authorized TLS certificates using the libp2p Auto-TLS client specification, rather than using self-signed ones.

This will enable protocol execution via web-transport and browsers for libp2p(-py).

PR: https://github.com/libp2p/py-libp2p/pull/1072
